# OASIS Skin Studio - Landing Page 🌿

Welcome to the repository for the **OASIS Skin Studio** website. This is a fast, responsive, and highly optimized static landing page built for a modern cosmiatry and aesthetic clinic.

This project was developed as a freelance solution to provide the business with a professional digital presence and a streamlined booking funnel.

Check out the live website here: [<img src="./img/favicon.ico" width="15px" height="15px"> OASIS Skin Studio](https://edinsonsan.github.io/oasis-skin-studio/)

## 📖 Description

The main objective of this website is to showcase the clinic's catalog of services (facials, body treatments, and micropigmentation) and convert visitors into clients through a frictionless booking process.

**Primary Goal:** To drive targeted traffic directly to the clinic's WhatsApp Business for appointment reservations.

### ✨ Key Features

- **Decoupled Architecture:** Header and Footer are modularized using Vanilla JavaScript (`components.js`) to keep the HTML clean and maintainable without needing a backend.
- **Dynamic WhatsApp Integration:** Call-to-action buttons generate customized, pre-filled WhatsApp messages based on the selected treatment.
- **Interactive UI:** Custom JavaScript logic for toggling between treatment categories ("For Women" / "For Men") with interactive data points.
- **Modern CSS Branding:** Global CSS variables applied for consistent "Lavender & Calm" branding, moving away from bloated template styles.

## 🛠️ Tech Stack & Dependencies

- **Core:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **UI Framework:** Bootstrap (via CDN for fast loading)
- **Typography:** Google Fonts (`Playfair Display` for elegant headings, `Lato` for readability)
- **Sliders:** Swiper.js / OwlCarousel
- **Icons:** FontAwesome
